### What I learned

To center elements on the page, I learned how to use Flexbox on the body:

```css
body {
	display: flex;
	justify-content: center;
	align-items: center;
	height: 100vh;
}
```

I also learned the difference between font-weight (for bold text) and font-style (for italic text).
